public MongoDB\WriteResult
# bulkWrite( array $bulk, array $options = array() )

Adds a full set of write operations into a batch and executes it

Adds a full set of write operations into a batch and executes it

The syntax of the $bulk array is: $bulk = [ [ 'METHOD' => [ $document, $extraArgument1, $extraArgument2, ], ], [ 'METHOD' => [ $document, $extraArgument1, $extraArgument2, ], ], ]

Where METHOD is one of - 'insertOne' Supports no $extraArgument - 'updateMany' Requires $extraArgument1, same as $update for MongoDB\Collection\updateMany() Optional $extraArgument2, same as $options for MongoDB\Collection\updateMany() - 'updateOne' Requires $extraArgument1, same as $update for MongoDB\Collection\updateOne() Optional $extraArgument2, same as $options for MongoDB\Collection\updateOne() - 'replaceOne' Requires $extraArgument1, same as $update for MongoDB\Collection\replaceOne() Optional $extraArgument2, same as $options for MongoDB\Collection\replaceOne() - 'deleteOne' Supports no $extraArgument - 'deleteMany' Supports no $extraArgument
